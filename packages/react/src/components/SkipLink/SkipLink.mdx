# SkipLink

Vi använder genvägskomponenten `SkipLink` när vi vill hjälpa personer som endast använder tangentbordet för att navigera, så att de enkelt kan hoppa till det viktigaste innehållet på en sida. Genvägen bör vara det första elementet markören går till när du navigerar med tangentbordet.

<Primary />
<Controls />

## Användning

```tsx
import { SkipLink } from '@digdir/designsystemet-react';

<SkipLink href='#main-content'>Hoppa till huvudinnehållet</SkipLink>

<main id='main-content' tabIndex={-1}>
  Region som kan ta emot fokus från skiplink.
</main>
```

## Riktlinjer för `SkipLink`

Vissa användare navigerar med Tab-tangenten på tangentbordet när de ska flytta markören förbi länkar eller formulärelement på en webbplats. När vi inkluderar en genvägskomponent på sidan får dessa användare möjlighet att hoppa över alla navigeringslänkar högst upp på sidan och gå direkt till huvudinnehållet.

Det är vanligtvis ett krav att ha med en genvägskomponent i fasta sidmallar för alla webbplatser.

Om det är färre än fem Tab-stopp till huvudinnehållet kan det hända att du inte behöver lägga till en `SkipLink`, eftersom det bara skulle tillföra fler knappar att hantera.

Vi placerar genvägen som det första fokuserbara elementet. Om sidan har en cookie-meddelande, lägger du `SkipLink` precis efter koden för detta meddelande.

Genvägen bör tydligt sticka ut från resten av innehållet, så att personer som kan se men använder tangentbordet för att navigera känner igen detta element innan de flyttar vidare.

<br />

## Text i komponenten

Se till att texten tydligt anger vad genvägen ska användas till. Använd enkel text, som till exempel "Gå direkt till innehållet" eller "Hoppa direkt till innehållet".

<br />

## Tillgänglighet

Genvägskomponenten är vanligtvis inte synlig förrän ett tryck på tangentbordet aktiverar den.

- Genvägen måste bli tydligt synlig när den får fokus.
- Texten på genvägen måste ha rätt färgkontrast.

<br />

Observera att vissa automatiska tester för tillgänglighet kan varna om att genvägen saknar ett landmärke. Denna varning gäller inte genvägar, så du kan ignorera den.

Se till att genvägar beter sig förutsägbart. En genväg med texten "Gå direkt till innehållet" ska flytta användarens fokus till början av innehållet, inte till början av det första stycket under rubriken.
