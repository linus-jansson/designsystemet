# Badge

`Badge` är en icke-interaktiv komponent som visar status med eller utan siffror.

<Primary />
<Controls />

## Användning

```tsx
import { Badge } from '@digdir/designsystemet-react';

/* Inline */
<Badge count={5} />

/* Flytande över element */
<Badge count={5}>
  <Elem />
</Badge>

/* Flytande över element som dot */
<Badge>
  <Elem />
</Badge>
```

## Flytande över element

Ställ in placering med `placement`. Om elementet du placerar över är en cirkel kan du sätta `overlap="circle"`.

<Canvas of={BadgeStories.Floating} />

Ibland kan ikoner vara olika, och då måste du ändra var `Badge` ska placeras.
Om du inte får `Badge` att ligga där du vill kan du använda `style` för att åsidosätta positioneringen.
Detta fungerar även med `overlap="circle"`.

<Canvas of={BadgeStories.CustomPlacement} />

## Visa status

Om du inte skickar in `count` visas endast en cirkel. Detta kan användas för att visa status.

<Canvas of={BadgeStories.Status} />

Du kan använda komponenten inline eller flytande över ett element.
