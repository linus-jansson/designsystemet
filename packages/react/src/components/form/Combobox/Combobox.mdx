## I formulär

Du måste lägga till `name` på `Combobox` för att den ska fungera i ett formulär.
Den använder inbyggda `select`-element under huven, så du kan använda
`Combobox.Option` som vanligt.

## Virtualiserad

Listan med element kan bli väldigt lång, så det kan vara bra att virtualisera.
Det gör du genom att sätta `virtual` till `true` på `Combobox`.

## `Combobox.Option`

Används som direkt barn av `Combobox` för att visa ett objekt i listan.
Texten från `children` visas för användaren när objektet är valt. `value` är
det som skickas till `onValueChange` när objektet blir valt.

<ArgTypes of={Combobox.Option} />

## `Combobox.Empty`

Används som direkt barn av `Combobox` för att visa ett meddelande när det inte finns
några objekt i listan.
