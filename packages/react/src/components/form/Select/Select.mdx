# Select

`Select` är en enkel nedrullningslista som låter användaren välja ett eller flera alternativ från ett statiskt urval.
Komponenten är byggd på HTML-elementet `select`, som använder sig av operativsystemets standarder (därav "native").
Detta säkerställer förutsägbarhet och tillgänglighet, men det innebär vissa begränsningar för hur komponenten kan anpassas.
Till exempel är det inte möjligt att åsidosätta formateringen av alternativen.

## Användning

```tsx
import { Select } from '@digdir/designsystemet-react';

<Label htmlFor='my-select'>Gör ett val</Label>
<Select id='my-select'>
  <Select.Option value='1'>Alternativ 1</Select.Option>
  <Select.Option value='2'>Alternativ 2</Select.Option>
  <Select.Option value='3'>Alternativ 3</Select.Option>
</Select>;
```
