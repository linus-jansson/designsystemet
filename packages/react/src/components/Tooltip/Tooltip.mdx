# Tooltip

`Tooltip` brukes for å vise korte forklaringer til brukeren. 

<Primary />
<Controls />

## Så här använder du `Tooltip`

Du kan antingen skicka in en `string` eller ett element som `children`. Om du skickar in en `string`, kommer den att placeras i ett `<span>`.

```tsx
import { Tooltip } from '@digdir/designsystemet-react';

// bara string, renderar ett `span`
<Tooltip content="Visas i tooltip">
  Trigger
</Tooltip>

// med element, renderar ditt element
<Tooltip content="Visas i tooltip">
  <MyButton>Trigger</MyButton>
</Tooltip>
```
