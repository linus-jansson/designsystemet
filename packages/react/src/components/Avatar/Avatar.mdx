# Avatar

`Avatar` är en komponent som visar en bild, initialer eller ikon för en användare eller profil.

<Primary />
<Controls />

## Användning

```tsx
import { Avatar } from '@digdir/designsystemet-react';

/* Med standard användarikon */
<Avatar aria-label="Ola Nordmann" />

/* Med initialer */
<Avatar aria-label="Ola Nordmann">ON</Avatar>

/* Med bild */
<Avatar aria-label="Ola Nordmann">
  <img src="..." />
</Avatar>
```

## Standardikon

Om du inte skickar in `children` visar vi en standard användarikon.

<Canvas of={AvatarStories.NoName} />

## Storlekar

`Avatar` kommer i flera storlekar. Text och ikon skalas, men ditt ikon kan behöva anpassas.

<Canvas of={AvatarStories.Sizes} />

## Färgvarianter

Du kan använda alla dina temafärger på `Avatar`.

<Canvas of={AvatarStories.ColorVariants} />

## Varianter

Du kan växla mellan cirkulär och fyrkantig form. Standard är cirkulär.

<Canvas of={AvatarStories.ShapeVariants} />

## Med bild

Om du vill ha en bild, lägg in den som ett direkt barn till `Avatar`.

Bilden (och andra child-element) får automatiskt `aria-hidden="true"` för att undvika dubbel information.

<Canvas of={AvatarStories.WithImage} />

## Komponering

Du kan komponera `Avatar` in i andra komponenter, som `DropdownMenu`, samt använda komponenter som
`Badge` runt om för att visa status.

<Canvas of={AvatarStories.InDropdown} />

## Som länk

Lägg länkar eller knappar runt `Avatar` för att göra det klickbart.
Du måste själv ställa in styling om du ska ha en interaktiv `Avatar`.
