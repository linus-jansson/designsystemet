# Skeleton

`Skeleton` kan användas för att indikera att innehåll på en sida laddar. Till skillnad från `Spinner` kan `Skeleton` ge en indikation till användaren om hur sidan kommer att se ut.

<Primary />
<Controls />

## Användning

<Canvas of={SkeletonStories.Components} />

Skeleton kommer i tre olika varianter: `Skeleton variant="rectangle"`, `Skeleton variant="circle"`, och `Skeleton variant="text"`.
Du kan bygga upp komponenter och sektioner av din sida genom att använda dessa som byggstenar. Målet är att resultatet ska efterlikna det innehåll som laddas.

<Canvas of={SkeletonStories.UsageExample} />

### Skalning av komponenten

Alla varianter av skeleton har `height` och `width` props, som kan användas för att manuellt ställa in storlekar. Du kan ange storlekarna i `px`, `%`, eller andra enheter som kan ställas in direkt på stil.
För `Skeleton variant="text"` räcker det ofta med att bara ställa in `width` till det antal bokstäver du förväntar dig ska visas, då höjden automatiskt skalas efter textinnehållet.

Skeleton anpassar sig också efter `children` som du skickar in till komponenten, men detta resulterar i en höjd och bredd som matchar summan av alla barn.

<Canvas of={SkeletonStories.Children} />

## Text

`Skeleton variant="text"` skalas automatiskt efter den lokala teckenstorleken, oavsett om den kommer från `parent`, `children`, eller för att typografikomponenter har ställts in på Skeletons genom `asChild`-propen.
För bästa resultat rekommenderar vi att du använder flera `Skeleton variant="text"` med `width` inställd på det antal bokstäver du förväntar dig ska visas.
